# Comment - effacer ou réinitialiser une NVRAM :
```

-éteindre le serveur
-débrancher les prises d'alimentation
-ouvrir le serveur
 Changez maintenant la position 6 du commutateur de maintenance du système de la position par   défaut "off" à "on" 
 ```
